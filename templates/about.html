About This Project

UPES Major Project: Spend Analyzer

This project aims to provide a comprehensive analysis of financial transactions by merging multiple CSV files and generating interactive visualizations. The tool helps users identify spending trends, categorize expenses, and analyze monthly expenditures across various accounts.

Objective

The primary goal is to develop a dynamic dashboard for students and professionals to manage and analyze transactional data, ensuring user-friendly insights and accuracy in financial monitoring.

Features

  • Upload multiple CSV files.
  • View categorized and monthly spending trends.
  • Identify high-spending categories and transaction distributions.
  • Interactive charts and visualizations for better insights.
