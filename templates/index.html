About the Project

The Spend Analyzer Dashboard is designed to help users track and analyze their financial transactions effectively. Upload your CSV files containing transaction details, and the dashboard generates visualizations such as spending trends, transaction categories, and monthly summaries. Gain insights to make informed financial decisions and save money.

Features include:

  • Detailed spending analysis by category and account
  • Interactive charts to visualize transaction data
  • Downloadable merged statements
  • Monthly and daily transaction summaries
